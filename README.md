Что внутри этого EXE (полный список функций)
Как ты и просил — всё в одном флаконе:

🤖 Отключение ИИ и Copilot: Убираем кнопку Copilot, удаляем пакеты AIX/Recall, отключаем веб-поиск.

🔒 Тотальная Приватность: Вырезаем телеметрию, блокируем сервера Microsoft через hosts, отключаем рекламный ID и Cortana.

🚀 Режим Максимум: Активируем скрытую схему питания "Ultimate Performance", заставляем CPU работать на 100%, отключаем гибернацию и энергосбережение PCI-E.

🧹 Убийца Блотварета: Удаляем встроенные Xbox-сервисы, рекламные плитки, ненужные фоновые приложения.

💾 Очистка Системы: Удаление Temp, кэша DNS, Prefetch и мусора из браузеров (Chrome/Firefox).

🎨 Отключение Анимаций: Выключаем визуальные эффекты для максимальной плавности на слабых ПК.
